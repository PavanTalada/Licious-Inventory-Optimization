# Licious Fresh Inventory Analytics

### 📌 Project Overview
This project solves the **High-Perishability** problem for Licious. In the fresh meat industry, items have a 24-48 hour shelf life. This tool uses the **Newsvendor Model** to calculate the mathematical 'sweet spot' between wasting stock and losing customers.

### 🛠 Tech Stack
- **Python** (Pandas, Scipy, Numpy)
- **Streamlit** (UI Dashboard)
- **Matplotlib** (Visualization)

### 🚀 How to Run
1. `pip install streamlit pandas scipy matplotlib`
2. `streamlit run app.py`
